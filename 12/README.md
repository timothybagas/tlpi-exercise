# Chapter 12

The `/proc` file system exposes a range of kernel informatin to application programs.

## Exercises

### 12.1

Program to lists the process ID and command name or all processes being run by the user named in 
the program's command-line argument.
> Implemented in `121/`

### 12.2

Program that draws a tree showing the hierarchical parent-child relationships of all processes 
on the system, going all the way back to _init_. Each process should display the process ID and 
and the command being executed.
> Implemented in `122/`

### 12.3

Program that lists all processes that have a particular file pathname open.
> Implemented in `123/`
